# {frontmatter.title}

## CoreLinux Fundamentals

- Filesystem structure — /etc, /var, /home, /tmp, /bin and what lives where.
- File permissions — chmod, chown, rwx, understanding users and groups.
- Essential commands — ls, cd, cp, mv, rm, find, grep, cat, tail, less.
- Process management — ps, top, htop, kill, systemctl.
- Package management — apt (Debian/Ubuntu), yum/dnf (RHEL/CentOS).
- Text editors — basics of nano, awareness of vim.
- Shell scripting basics — variables, loops, conditions in bash.
- Logs — reading /var/log/, using journalctl.
- Cron jobs — schedule recurring tasks with crontab.
- SSH — connect, manage keys, scp to transfer files.

## Networking on Linux

- ip, ifconfig — network interface management.
- netstat, ss — view open ports and active connections.
- ping, traceroute, nslookup, dig — diagnose network issues.
- iptables / ufw — basic firewall management.
- tcpdump — capture and inspect network traffic.

## Tools

- VirtualBox — Run Linux VMs locally for practice.
- Ubuntu Server — Best distro to start with in a lab environment.
- Debian — Stable, widely used in enterprise environments.
