## Windows Server Fundamentals

- Roles and Features — what they are and how to install them (DNS, DHCP, IIS, File Server).
- Active Directory Domain Services (AD DS) — the heart of Windows infrastructure.
- Group Policy (GPO) — deploy settings and restrictions across machines.
- DNS — name resolution, forward/reverse zones, records (A, CNAME, MX, PTR).
- DHCP — IP address assignment, scopes, reservations.
- File sharing — NTFS permissions vs share permissions, understanding the difference.
- Remote Desktop (RDP) — connect to and manage remote machines.
- Windows Event Viewer — read logs, identify errors and warnings.
- Task Scheduler — automate recurring tasks on Windows.
- PowerShell basics — get comfortable running and reading scripts.

## Active Directory

- Users and computers — create, modify, disable accounts.
- Organizational Units (OU) — structure and delegate administration.
- Security Groups vs Distribution Groups — know the difference and when to use each.
- GPO linked to OUs — apply policies to specific users or machines.
- gpupdate /force, gpresult — apply and troubleshoot group policies.
- Password policies — complexity, expiration, lockout settings.

## Tools

- Windows Server 2019/2022 — Set up a lab VM and practice.
- RSAT — Remote Server Administration Tools, manage AD from your workstation.
- Sysinternals Suite — Microsoft toolkit for advanced Windows diagnostics.
