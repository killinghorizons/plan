## Network Fundamentals

- OSI Model — understand the 7 layers and what happens at each one.
- TCP/IP stack — how data travels from app to wire.
- IP addressing — IPv4, subnet masks, CIDR notation (/24, /16 etc.).
- Subnetting — calculate network ranges, broadcast address, usable hosts.
- Key protocols — TCP vs UDP, HTTP/HTTPS, DNS, DHCP, FTP, SSH, ICMP.
- MAC addresses — what they are, ARP protocol.
- Default gateway, routing basics — how packets find their destination.
- VLANs — segment a network logically on a switch.
- NAT — how private IPs reach the internet.
- Firewalls — stateful vs stateless, rules, zones.

## Tools

- Cisco Packet Tracer — Simulate networks without physical hardware, free with Cisco NetAcad account.
- GNS3 — More advanced network simulator, closer to real hardware.
- Wireshark — Capture and analyze real network traffic, essential tool.
- Nmap — Network scanner, discover hosts and open ports.
