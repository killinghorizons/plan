## Language Fundamentals

- Variables, types, pointers, references — the unavoidable foundation of C++.
- Manual memory management — new, delete, understanding stack vs heap.
- Functions, pass by value / reference / pointer.
- Structures (struct) — sufficient for your level, no need to deep dive into classes.
- Header files .h and code separation.

## Essential Stdlib

- [iostream](https://cppreference.com/cpp/header/iostream) — Console input/output (cin, cout).
- [fstream](https://cppreference.com/cpp/io/basic_fstream) — Read and write files.
- string — String manipulation.
- vector — Dynamic array, the most commonly used container.
- map — Key/value storage.
- algorithm — Sort, search and manipulate collections.
- cstdlib — Basic system functions.

## Network

- Winsock2 (winsock2.h) — Windows equivalent of Python's socket module, low-level network communication.
- TCP/UDP sockets in C++ — more verbose than Python but important to understand.

## To Explore Later

- thread — Multi-threaded programming
- Boost.Asio — Asynchronous networking, the C++ equivalent of asyncio
- Qt — Build graphical user interfaces in C++
- OpenSSL — Encryption and SSL in C++, widely used in production environments
